ANSWERS TO RESEARCH QUESTIONS
================================================================

Q1: Does generate() provide a way to generate ID data?
A1: YES ✅
    - Automatic UUID v4 generation (uuid.uuid4().hex)
    - Returned in every response via meta_info["id"]
    - Works for single, batch, and streaming requests
    - Implementation in io_struct.py and tokenizer_manager.py

Q2: Are there any chat templates?
A2: YES ✅ - TWO COMPLETE SYSTEMS
    
    System 1: Frontend Language Chat Templates
    - File: python/sglang/lang/chat_template.py
    - 20+ pre-configured templates
    - Models: Llama, Qwen, Mistral, DeepSeek, Vicuna, Gemma, etc.
    - Auto-selection based on model path
    
    System 2: OpenAI-Compatible API Chat Templates
    - File: python/sglang/srt/conversation.py
    - Supports Hugging Face tokenizer templates
    - Customizable via JSON/Jinja formats
    - Used with /v1/chat/completions endpoint
